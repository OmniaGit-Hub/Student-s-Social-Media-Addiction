# Student-s-Social-Media-Addiction
Built interactive Power BI dashboard analyzing social media usage patterns across 500+ student records; designed 4 KPIs segmented by demographics and usage frequency to surface behavioral insights .
# The project is divided into two dashboards:
Dashboard 1 – Overview: High-level insights and global patterns
Dashboard 2 – Behavior & Impact: Deep-dive analysis into behavior, platforms, and impacts
# Objectives
Measure the average addiction level among students
Understand how addiction affects:
Mental health
Sleep hours
Academic performance
Compare addiction levels across:
Gender
Country
Social media platforms
Relationship status
 # Tools & Technologies
Power BI Desktop
Power Query (data cleaning & transformation)
DAX (measures and KPIs)
How to Use
# Download the .pbix file
Open it using Power BI Desktop
Use filters on Dashboard 1 to control both dashboards
# Key Insights 
Higher addiction scores are associated with lower mental health scores
Students with high addiction tend to sleep fewer hours
Certain platforms show significantly higher addiction levels
